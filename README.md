# Neodent CNC

Cockpit operacional CNC modular.

## Páginas

- `index.html` — Portal
- `login.html` — Login
- `admin.html` — Administração de usuários e permissões
- `painel.html` — Painel geral por cards
- `tempo-linha.html` — Tempo da Linha
- `preparadores.html` — Atualizações operacionais
- `apoio.html` — Pedidos de ajuda
- `setup.html` — Setups e liberação com ciclo 100%
- `historico.html` — Histórico por TNL
- `relatorio.html` — Relatório automático

## Estrutura

CSS e JS ficam separados em `assets/`.

## API

A API base fica em `assets/js/core/config.js`.
